## ğŸš€ Getting Started

### Prerequisites
- Node.js (v18+)
- npm

### Installation

1.  **Clone the repository**
    ```bash
    git clone https://github.com/thisisniagahub/QuranPulse-v6.0.git
    cd QuranPulse-v6.0
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Run the development server**
    ```bash
    npm run dev
    ```
    Visit `http://localhost:5173` to view the app.
